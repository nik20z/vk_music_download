Version: python-3.8.5 (64-bit)

Скрипт предназначен для скачивания музыки из социальной сети Вконтакте. Бот принимает сообщения и сохраняет вложенные аудио. Для ускорения процесса используются потоки.

Настройка:
1. Создать группу и получить token и id группы.
2. Внести изменения в файл settings.txt, заменив предложенные значения на свои:
обязательные поля: token,  group_id;
audio_format - формат, в котором необходимо сохранить аудио;
path - абсолютный путь к папке для сохранения)
